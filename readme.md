# React Laravel chatting realtime app
## Installer
- Step 1: Clone this repository
- Step 2: Composer install
- Step 3: Npm install
- Step 4: npm run dev
- Step 5: php artisan migrate
- Step 6: php artisan db:seed
- Step 7: run app...
